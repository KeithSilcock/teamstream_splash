Student Grade Table

A Content Management System for students' grades

This application was built using ReactJS and ReduxJS for the front end, and has a PHP backend.

Building this application was a great opportunity for me to fully understand how to build both the Front and Back end. ReduxJS was the biggest challenge to understand, but by utilizing it in this application, it helped me understand to a point where I can teach it to others.
